# Prompty - Senior AI Product Architect

Prompty is a high-fidelity, typography-first SaaS platform designed to transform vague product ideas into structured, professional-grade prompt systems for AI coding tools like Cursor, Bolt, and Lovable.

Built with a "Senior Architect" persona, Prompty guides users through a multi-step builder to define technical architecture, constraints, and features, generating modular prompts that minimize AI hallucinations.

## ✨ Key Features

- **Multi-Step Builder:** 11-step interactive flow for deep product definition.
- **Gemini AI Integration:** Powered by Google's Gemini 1.5 Flash for intelligent prompt refinement.
- **Senior Persona:** Generates prompts from the perspective of a Principal Engineer.
- **Minimalist UX:** Focus on typography, whitespace, and clean vertical flow.
- **Privacy First:** local-first processing with no server-side data storage.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- Google Gemini API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/prompty.git
   cd prompty
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory.
   - Add your Gemini API key:
     ```env
     VITE_GEMINI_API_KEY=your_gemini_api_key_here
     ```

4. Start the development server:
   ```bash
   npm run dev
   ```
